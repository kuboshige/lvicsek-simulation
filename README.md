# lvicsek-simulation

**Vicsek モデル**（自己推進粒子の集団運動）をブラウザで体験できるインタラクティブ可視化。

`index.html` をブラウザで開くだけ（依存なし・単一ファイル）。

## 遊び方
- 最初は **1点** がひゅんひゅん漂っているだけ。
- 画面を **タップ／ドラッグ** すると粒子がどんどん増える。
- 数が **臨界点** を超えると、ばらばらだった動きが一斉にそろい、
  **群れ**（フロッキング）へと相転移する。
- 色 = 進行方向。バラバラの虹色 → 一色にそろう、で秩序化が一目でわかる。

右上の ⚙ からノイズ η・速さ v・視野半径 r を調整可能。
`Space` で中央に一気に追加、`R` / `C` で1点に戻す。

## 数理（2D 版）
各粒子は位置 **r**ᵢ と向き Θᵢ を持ち、速さ v は一定。近傍半径 r 内の平均向きに
ノイズ η を加えて更新する。

- 方向更新: Θᵢ(t+Δt) = arg( Σ_{|rᵢ−rⱼ|<r} e^{iΘⱼ(t)} ) + ηᵢ(t)
- 位置更新: **r**ᵢ(t+Δt) = **r**ᵢ(t) + v·Δt·(cosΘᵢ, sinΘᵢ)
- 秩序パラメータ: v_a = (1/N)·|Σ **s**ᵢ|  （v_a ≈ 1 で強い集団運動）

境界は周期境界（トーラス）、近傍探索はセルリスト（格子）で高速化している。
